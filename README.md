# TIMEFMT

**Timestamp Format Converter**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/timefmt.git
cd timefmt
go build -o timefmt .
```

## Usage

```bash
./timefmt --help
```
