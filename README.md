## Timefmt

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/timefmt.git
cd timefmt
go build -o timefmt .
```

### Usage

```bash
./timefmt --help
```

---

## Quick Start

```bash
./timefmt [command]
```
