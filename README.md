## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/timefmt.git
cd timefmt
go build -o timefmt .
./timefmt --help
```

## 🚀 Quick Start

```bash
./timefmt -h
```

## 🧪 Examples

```bash
# Example 1
./timefmt status

# Example 2  
./timefmt --verbose

# Example 3
```
